# WiFi Scanner App

## Project Overview
A simple Android application for scanning and displaying nearby WiFi networks.

## Development Setup
1. Ensure you have Android Studio installed
2. Clone the repository
3. Open the project in Android Studio

## Running Tests
### Unit Tests
```bash
./gradlew test
```

### Instrumentation Tests
```bash
./gradlew connectedAndroidTest
```

## Key Components
- WiFi Network Scanning
- RecyclerView for Network Display
- Material Design Components
